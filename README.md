# The way of the Rex

Стартовый каркас проекта для игры на Python и PyGame.

## Идея

Маленький динозавр выживает в опасном доисторическом мире, собирает еду, растёт и постепенно проходит более сложные уровни.

## Первый этап

- Создана базовая структура проекта.
- Подготовлены зависимости.
- Добавлена точка входа для будущего игрового цикла.

## Второй этап

- Добавлено окно PyGame и основной игровой цикл.
- Появился отдельный класс `Game`.
- Управление выходом теперь работает через `ESC` и закрытие окна.

## Третий этап

- Добавлен отдельный класс `Player`.
- Появилось управление стрелками и `WASD`.
- Динозавр теперь двигается по экрану и не выходит за границы окна.

## Четвёртый этап

- Добавлен отдельный класс `Food`.
- На экране появилась еда, которую можно собирать.
- После сбора еды динозавр растёт, а счёт увеличивается.

## Пятый этап

- Добавлен класс `Enemy` и простая логика их движения.
- Добавлена система жизней: при столкновении с врагом жизни уменьшаются.
- При падении жизней до нуля игра останавливается (пока простая логика Game Over).
